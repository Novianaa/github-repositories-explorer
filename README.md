<!-- GETTING STARTED -->
## Getting Started<hr />

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Novianaa/github-repositories-explorer.git
   ```
2. Install NPM packages
   ```sh
   npm install / yarn install
   ```
3. Then, you can running this project with the command below
   ```sh
   npm start / yarn start
   ```
